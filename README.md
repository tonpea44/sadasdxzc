
# ระบบแชร์สูตรอาหารภาษาไทย

## โฟลเดอร์

- `client/` - React + Tailwind + Vite (หน้าเว็บ)
- `server/` - Express API (ฝั่ง Backend)

## วิธีติดตั้ง

```bash
cd client
npm install
npm run build
```

```bash
cd ../server
npm install
node server.js
```

## การ Deploy
- frontend → Netlify
- backend → Render
